Close

This is a 1v1 two player shooter with basically two simple actions :
- Jump and dodge incoming damage.
- Shoot some damage from a random range of 1-5 to the opponent.

The key mapping is very simple.
Player one can use keyboard based keys where :
- "W" is for dodging / jumping.
- "S" for shooting the damage to the opponent
Player two can use keyboard based keys where :
- "Up Arrow" is for dodging / jumping.
- "Down Arrow" for shooting the damage to the opponent

THIS GAME IS NOT MOBILE RESPONSIVE AND IS ONLY IN IT'S BETA STAGE
